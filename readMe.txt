How to run my script
Open and import the whole project in Intellij	
Just run the HW1Main.class
Because the path that I used is the relative path, you don't need to change the path 
And you need to follow the path in the Path.java
Add new folder under the src folder and create two folders input and output
output folder should have docset.trectext and docset.trecweb
input folder should have stopword.txt
Result will appear in the input folder
